## 1. 리눅스의 개요

### 1.1 리눅스의 탄생

- **유닉스 Unix**
    - 리눅스가 탄생하기 이전 운영체제

    - 통상적 리눅스 - 토르발스가 만든 커널에 컴파일러, 셀, 기타 프로그램이 조합된 배포판
    - 대표적인 배포판 - 우분투 리눅스

    ### 1-3 커널

    - **커널의 구성**
        - 안정버전 stable version

            이미 검증된 개발 완료 코드로 구성

        - 메인 라인 버전 mainline version

            토르발스가 개발 중인 버전

            안정 버전이 나오기 전에 추가된 기능을 미리 접하고 싶을 때 사용

        - 프리패치 버전 prepatch version

            안정 버전이 나오기 전에 추가된 기능을 미리 접하고 싶을 때 사용

    - 파일 이름에 붙은 숫자 **5.1.15의 5는 주버전, 1은 부버전, 15는 패치 버전**

## 3. 가상머신의 개요

### 3.1 가상머신의 개요

- **가상머신**

    PC 1대만으로 여러 대의 PC를 운영하는 것처럼 만드는 방법

- 실습에서 사용할 환경

    PC 1대에서 리눅스 3대 구동 (리눅스 서버 A, 리눅스 서버 B, 리눅스 클라이언트)

    4개의 운영체제가 PC 1대에서 가동

### 3-2 가상머신 소프트웨어의 개념

- 가상머신 소프트웨어

    컴퓨터에 설치된 운영체제 (호스트 OS) 안에 가상의 컴퓨터를 만들고

    이 가상의 컴퓨터 안에 또 다른 운영체제 (게스트 OS)를 설치 · 운영할 수 있도록 제작된 소프트웨어

- **호스트 OS** host operating system

    PC에 설치되어 있는 윈도우

- **게스트 OS** guest operating system

    가상머신에 설치한 운영체제

- 일반적인 환경

    호스트 컴퓨터에는 윈도우가 설치되어 있고,

    기존에 설정된 IP 주소를 이용하여 네트워크 카드를 통해 인터넷에 접속 가능

- 가상머신 실습 환경

    가상머신(게스트 컴퓨터) 3대를 만들고 리눅스 3대 설치

    가상머신 3대에는 가상 IP 주소 할당

    가상 라우터를 통해 가상머신 3대를 하나의 네트워크로 묶어 동일한 네트워크 환경으로 사용

### 3.3 가상머신 소프트웨어 종류

- 대표적인 가상머신 소프트웨어

    VMware Workstation Pro           유료

    VMware Workstation Player       무료
