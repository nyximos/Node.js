# #3 연산자 Operator

### 1. 연산자

- **연산자**
    - 어떠한 기능을 수행하는 기호
    - +, -, *, / 등
- **피연산자**
    - 연산자의 작업 대상
    - 변수, 상수, 리터럴, 수식

### 2. 연산자의 종류

- 단항 연산자
    - 필요한 연산자가 1개
    - + 양수
    - - 음수
    - ++
    - - -
    - ~
    - !
- **이항 연산자**
    - 필요한 연산자가 2개
    - **산술** + - * / % << >> >>>
    - **비교** > < ≥ ≤ == ≠
    - **논리** ? :
- **삼항 연산자**
    - 필요한 연산자가 3개? :
- **대입 연산자** =

### 3. 연산자의 우선순위

![https://s3-us-west-2.amazonaws.com/secure.notion-static.com/a0f121bf-2670-4718-a4ab-821daaeac3e4/Untitled.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/a0f121bf-2670-4718-a4ab-821daaeac3e4/Untitled.png)

- 괄호의 우선순위가 제일 높다.
- 산술 > 비교 > 논리 > **대입**
- 단항 > 이항 > 삼항
- 연산진행방향
    - 왼쪽에서 오른쪽 →
    - **단항**과 **대입**연산자는 오른쪽에서 왼쪽 ←
- << >> >>> 는 덧셈 연산자보다 우선순위가 낮다.
- ||와 |는 &와 &&보다 우선순위가 낮다.

### 4. 증감연산자 ++, - -

- **증가연산자 ++**

    피연산자의 값을 1 증가

- **감소연산자 - -**

    피연산자의 값을 1 감소

- 연산자의 위치에 따라 결과가 달라진다.
    - **전위형**

        값이 참조되기 전에 증가

    - **후위형**

        값이 참조된 후에  증가

### 5. 부호연산자 +,- 와 논리부정연산자!

- **부호연산자 + -**
    - **+** :  피연산자에 1을 곱하고
    - **-** :  피연산자에 1을 곱한다
- **논리부정연산자 !**

    true는 false로 false는 true로

    피연산자가 boolean일 때만 사용가능

### 6. 비트전환연산자 - ~

- 정수를 2진수로 표현했을 때,

    1을 0으로 0은 1로 바꾼다.

    ![https://s3-us-west-2.amazonaws.com/secure.notion-static.com/b32a1380-f30f-488e-965a-e1220aea4783/Untitled.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/b32a1380-f30f-488e-965a-e1220aea4783/Untitled.png)

### 7. 이항연산자의 특징

- 이항연산자는 연산을 수행하기 전에 피연산자의 타입을 일치시킨다.
- 피연산자중 표현범위가 큰 타입으로 형변환

    ![https://s3-us-west-2.amazonaws.com/secure.notion-static.com/748d038e-677e-403b-af11-4e9d92f58056/Untitled.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/748d038e-677e-403b-af11-4e9d92f58056/Untitled.png)

### 8. 나머지 연산자 %

- 나누기한 나머지 반환
- 홀수, 짝수등 배수검사에 주로 사용

### 9. 쉬프트 연산자 <<, >>, >>>

- 2^n으로 곱하거나 나눈 결과 반환
- 곱셈, 나눗셈보다 빠르다

![https://s3-us-west-2.amazonaws.com/secure.notion-static.com/57cd66a6-91d3-4ee1-b73e-3787804eb50f/Untitled.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/57cd66a6-91d3-4ee1-b73e-3787804eb50f/Untitled.png)

### 10. 비교연산자 >,<,≥,≤,==,≠

- 피연산자를 같은 타입으로 변환한 후에 비교한다.
- 기본형 boolean 제외
- 참조형 ==, ≠만 사용

![https://s3-us-west-2.amazonaws.com/secure.notion-static.com/bc44a92d-17a5-489c-b489-f5f450aa6a91/Untitled.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/bc44a92d-17a5-489c-b489-f5f450aa6a91/Untitled.png)

### 11. 비트연산자 &, |, ^

- 피연산자를 비트단위로 연산한다
- 실수형을 제외한 모든 기본형에 사용가능
- OR 연산자 **|**

    피연산자 중 어느 한 쪽이 1이면 1이다.

- AND 연산자 **&**

    피연산자 중 양 쪽 모두 1이면 1이다.

- XOR 연산자 **^ Exclusive OR 배타적OR**

    피연산자가 서로 다를 때 1이다.

### 12. 논리연산자 &&, ||

- 피연산자가 반드시 boolean
- 연산결과도 반드시 boolean
- &&가 ||보다 우선순위가 높다.
- OR 연산자 ||

    피연산자 중 한 쪽이 true이면 true

- AND 연산자 &&

    피연산자 중 양 쪽 모두 true이면 true

![https://s3-us-west-2.amazonaws.com/secure.notion-static.com/3d929b42-9275-4a00-94e9-245b6ee28f69/Untitled.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/3d929b42-9275-4a00-94e9-245b6ee28f69/Untitled.png)

### 13. 삼항연산자 ? :

- 조건식의 연산결과가 true이면 '식1'의 결과 반환
- false이면 '식2'의 결과 반환
- (조건식) ? 식1 : 식2

### 14. 대입연산자 =, op=

- 오른쪽 피연산자의 값을 왼쪽 피연산자에 저장
- 왼쪽 피연산자는 상수가 아니어야 한다.

```java
int i = 0;
i = i + 3;

final int MAX = 3;
MAX = 10; //에러
```
